## installation and setup

1. **Clone Repository**:

```bash
   git clone https://github.com/rajesh-code-dev/nodejs-backend.git
   cd nodejs-backend
```

2. **Install Dependencies**

   ```bash
      npm install
   ```

3. **Start Development Server**
   ```bash
      nodemon
   ```
